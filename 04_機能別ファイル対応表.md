# 機能別ファイル対応表 ― つづけるリハ

各機能が、どのファイル・どの関数で実現されているかの対応表。

| 機能 | 主担当ファイル | 主な関数 | 補足 |
|---|---|---|---|
| 画面の骨組み | index.html | ―（マークアップ） | 各ビュー・オーバーレイ・モーダルの器 |
| スタイル | style.css | ―（CSS） | 配色・カード・カレンダー・タイマー・編集UI |
| 共通処理 | helpers.js | `keyOf` `todayKey` `uid` `esc` | 日付キー・ID・エスケープ |
| 表示整形 | helpers.js | `units` `fmtDur` `freqText` `hasTimer` | 単位数・時間表記・頻度文・タイマー有無 |
| メニュー定義 | menu.js | `mkSlots` `mkSets` `mkTimer` `defaultMenu` `legacyMenu` | 既定は空、旧定義は過去日表示用 |
| 保存・読込 | menu.js | `loadMenu` `saveMenu` `loadData` `saveData` | localStorage 入出力 |
| 旧データ移行 | data.js | `isNewRec` `migrateRec` `migrateAll` | 旧形式→新形式 |
| 当日同期 | data.js | `snapMenu` `ensureToday` | 当日を現メニューへ／過去日は不変 |
| 集計 | data.js | `totalUnits` `doneUnits` `itemDone` `recComplete` | 達成率・完了判定の土台 |
| 今日ビュー描画 | today.js | `renderToday` `cardHTML` | セクション→カード描画・空状態ガイド |
| 達成記録 | today.js | `bindToday` `commitCard` | タップでトグル・部分更新 |
| 進捗・連続日数 | today.js | `updateProgress` `updateStreak` | リング・達成数・ストリーク |
| コンプリート演出 | today.js | `showComplete` `hideComplete` | 紙吹雪＋バイブ |
| 当日リセット | today.js | `resetToday` | 当日記録のみ消去 |
| タブ切替 | calendar.js | `switchView` | 今日／カレンダー |
| カレンダー | calendar.js | `renderCalendar` `moveMonth` | 月表示・色分け・月集計 |
| 日別詳細 | calendar.js | `openDay` `closeDay` | その日の内訳 |
| タイマー | timer.js | `buildPhases` `openTimer` `paintTimer` `tick` `toggleTimer` `closeTimer` | 単発／インターバル |
| 編集モード | editor.js | `toggleEdit` `renderEdit` `erowHTML` `bindEdit` | 一覧・各操作のひも付け |
| 項目の削除／セクション操作 | editor.js | `delSection` `moveSection` | セクション削除・上下移動 |
| 並べ替え（ドラッグ） | editor.js | `dragStart` `dragMove` `dragEnd` | セクション内スムーズ並べ替え |
| 項目エディタ | modal.js | `openModal` `updateModalFields` `saveModal` `closeModal` | 頻度＋独立タイマーの入力 |
| バックアップ／復元 | system.js | `exportData` `importData` | JSON書き出し・読み込み |
| 自動更新 | system.js | `checkUpdate` `autoUpdateCheck` | version.json 確認 |
| 初期化 | main.js | `setDateLine` `dismissHint` ＋起動処理 | 読込→移行→同期→描画→監視→SW登録 |
| オフライン・キャッシュ・更新基盤 | sw.js | ―（install/activate/fetch/message） | ネット優先・自動更新 |
| PWA設定 | manifest.json | ― | 名称・アイコン・表示形態 |
| 版数判定 | version.json | ― | 自動更新の比較対象 |

> 補足：JS は `index.html` 末尾で「helpers → menu → data → today → calendar → timer → editor → modal → system → main」の順に読み込み、グローバルスコープを共有する。
