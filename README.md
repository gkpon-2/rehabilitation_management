# つづけるリハ（PWA）

続けたいリハビリ・運動を自分で登録し、毎日タップで達成を記録できるPWAです。iPhoneのホーム画面に追加すると、アプリのように開けてオフラインでも動きます。初期状態はまっさらで、メニューはアプリ内の編集機能から作成します。

素材は商用素材・有料フォント・外部ライブラリを使用していません（アイコンは自作、項目アイコンは絵文字、フォントは端末標準、ライブラリなしの素のHTML/CSS/JS）。

## 入っているファイル
```
index.html        … アプリ本体（これ1つで動きます）
manifest.json     … ホーム画面追加の設定
sw.js             … オフライン用（サービスワーカー）
icons/            … アイコン画像
```
※ すべて同じ階層に置いてください。フォルダ構成は変えないでください。

---

## ① GitHub Pages で公開する手順

1. GitHub で新しいリポジトリを作る（例：`rehab`）。公開（Public）でOK。
2. 作成後の画面で **「uploading an existing file」** をクリック。
3. ここにある **index.html / manifest.json / sw.js と icons フォルダの中身** をまとめてドラッグ＆ドロップ。
   - icons の中の画像5つも忘れずに。GitHubの画面で `icons/icon-192.png` のようにパスを保ったままアップロードされればOKです。
4. 緑の **「Commit changes」** を押す。
5. 上のメニュー **Settings → 左メニュー Pages** を開く。
6. **Build and deployment** の Source を **「Deploy from a branch」**、Branch を **`main`／`/(root)`** にして Save。
7. 1〜2分待つとページ上部に公開URL（例：`https://ユーザー名.github.io/rehab/`）が出ます。

---

## ② iPhoneのホーム画面に追加（PWA化）

1. iPhoneの **Safari** で①の公開URLを開く。
2. 下の **共有ボタン（□に↑）** をタップ。
3. **「ホーム画面に追加」** を選ぶ → 右上の「追加」。
4. ホーム画面のアイコンから開けば、アドレスバーのない全画面アプリとして使えます。

> 記録は端末のSafari内に保存されます。日付が変わると自動で新しい一日に切り替わり、過去日の達成状況は連続記録（🔥）の計算に使われます。

---

## ③ メニューを自分で編集する（アプリ内でできます）

今日の画面の右上「✎ 項目を編集」を押すと編集モードになります。コードを触る必要はありません。

- **項目の追加**：各セクションの「＋ この欄に項目を追加」から。タイトル・アイコン・**頻度**（朝昼夜／朝夜／1日◯回／◯回（秒）×◯セット）・**タイマー**・メモを設定できます。
- **タイマーは頻度と独立**：どの頻度にも付けられます。「なし／つける（1回ぶんを計る・分秒）／つける（セットごとに計る・休憩あり）」から選択。例：「朝昼夜」＋「1回ぶん1分」で“1分立つ”、「20秒×10セット」＋「セットごと・休憩5秒」でキープ系が作れます。
- **項目の編集**：各項目の ✎ ボタン。
- **項目の削除**：各項目の 🗑 ボタン。
- **並べ替え**：各項目の右端の ≡ を押しながら上下にドラッグ（iPhoneのアラーム／ミュージック風に、行が浮いて他がよけます）。別のセクションへ移すときは ✎ →「セクション」で変更。
- **セクションの並べ替え**：セクション名の左の ▲▼ ボタン。
- **セクションの追加／名前変更／削除**：いちばん下の「＋ セクションを追加」、各セクション名は直接タップして書き換え、「削除」で削除。

**大事な仕様**：追加・削除・変更は **その日（今日）から** 反映されます。前日までのカレンダーの達成率は、当時のメニュー構成のまま固定され、変わりません。

## ④ アプリを更新したとき
コードを書き換えたら、次の3か所の数字をそろえて1つ増やしてから再アップロードしてください。
1. `index.html` の `APP_VERSION='6'`
2. `version.json` の `"version": "6"`
3. `sw.js` の `const CACHE = 'rehab-v6';`

アプリは「起動したとき」と「ホーム画面から前面に戻ったとき」に version.json を確認し、新しい番号を見つけると自動で再読み込みして最新になります。ホーム画面アプリの入れ直しは不要です（記録も残ります）。

## ⑤ 記録のバックアップ・復元（大切）
- iPhoneは **ホーム画面からアプリを削除すると、記録（保存データ）も一緒に消えます**。アンインストールや作り直しの前には必ずバックアップを。
- 画面下の **「書き出す（保存）」** でJSONファイルが保存できます（ファイルアプリやiCloudに置いておくと安心）。
- 別端末や入れ直しのあとは **「読み込む（復元）」** でそのJSONを選べば記録が戻ります。
- 自動更新に対応したので、基本的に **アプリを削除して入れ直す必要はありません**。そのまま使い続ければ記録は残ります。
